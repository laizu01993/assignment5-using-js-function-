Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document.

How do you select an element from the DOM?

document.querySelector('selector').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manupulate an element's attributes and styles using the DOM?

The onfocus event handler method is used to change the element's style when the user clicks inside it.
